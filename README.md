# Word Search Game

A cozy, browser-based word search game with classic and timed level challenge modes.

## Overview

- Generate fresh puzzles and hunt for hidden words by dragging across the grid.
- Switch between Classic mode and Level Challenge with a countdown.
- Track progress with a local leaderboard and celebratory confetti.

## Getting Started

```bash
npm install
npm run dev
```

Open the URL printed by Vite in your terminal.

## Scripts

- `npm run dev` - start the dev server
- `npm run build` - type-check and build for production
- `npm run preview` - preview the production build
- `npm run lint` - run ESLint

## Tech Stack

- React 19 + TypeScript
- Vite 7
- Tailwind CSS 4
